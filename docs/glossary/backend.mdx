<p className="cf-kicker">GLOSSARY · ARCHITECTURE</p>

The backend is the FastAPI service under `apps/api/`. It currently implements
only the identity bounded context.

Why it matters here: the backend replaced the frontend's mock auth and
preferences layer in Spec 002.

Where it appears: [Backend architecture](/get-started/architecture/backend), [Identity backend overview](/backend/overview).

Related terms: [FastAPI](/glossary/fastapi), [Bounded context](/glossary/bounded-context).
