<p className="cf-kicker">GLOSSARY · PERSISTENCE</p>

A database stores durable application state. Campfire uses PostgreSQL 16 for the
identity backend.

Why it matters here: users, credentials, preferences, sessions, and refresh
tokens persist outside the browser.

Where it appears: [Data and persistence](/backend/data-persistence), [ADR-0001](/sdd/adrs/0001-postgresql-as-engine).

Related terms: [Migration](/glossary/migration), [Repository pattern](/glossary/repository-pattern).
