FastAPI is the HTTP framework used by `apps/api/`.

Why it matters here: FastAPI owns routing, request/response validation, and
OpenAPI generation at the adapter edge.

Where it appears: [Backend architecture](/get-started/architecture/backend), [API endpoints](/backend/api-endpoints).
