<p className="cf-kicker">GLOSSARY · SPECS</p>

A contract describes how two parts of the system communicate. Campfire uses UI
contracts for frontend behavior and OpenAPI for backend HTTP behavior.

Why it matters here: Spec 002's OpenAPI snapshot protects the API shape used by
the frontend.

Where it appears: [Backend contracts](/backend/contracts), [Artifacts index](/sdd/artifacts/index).
