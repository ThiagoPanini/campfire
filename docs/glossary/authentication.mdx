<p className="cf-kicker">GLOSSARY · IDENTITY</p>

Authentication verifies a user's identity. Campfire currently supports
email/password authentication and a development-only Google stub.

Why it matters here: authentication creates server-side sessions and returns an
opaque access token plus refresh cookie.

Where it appears: [Auth flow](/backend/auth-flow), [Identity backend overview](/backend/overview).

Related terms: [Authorization](/glossary/authorization), [JWT](/glossary/jwt).
