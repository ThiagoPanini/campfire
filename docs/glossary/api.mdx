<p className="cf-kicker">GLOSSARY · BACKEND</p>

An API is the contract a client uses to interact with a service. Campfire's
current API is the identity backend exposed by FastAPI.

Why it matters here: the frontend calls the API for registration, login,
refresh, logout, current-user data, and preferences.

Where it appears: [API endpoints](/backend/identity/api-endpoints), [Backend contracts](/backend/identity/contracts).

Related terms: [Endpoint](/glossary/endpoint), [OpenAPI](/glossary/openapi).
