This quickstart gets the current Campfire local workflow running with the same command surface used by CI. For the full platform guide, see [Local environment](/guides/platform/local-environment).

## Prerequisites

<Steps>
  <Step title="Install runtimes">
    - Python 3.12
    - [uv](https://docs.astral.sh/uv/) for Python dependency management
    - Node.js 20 LTS and npm for the frontend
    - Docker with Docker Compose v2 for LocalStack
    - GNU Make
  </Step>
  <Step title="Clone and enter the repository">
    ```bash
    git clone https://github.com/paninit/campfire.git
    cd campfire
    ```
  </Step>
</Steps>

## Check your machine

```bash
make doctor
```

<Check>
The command reports missing tools with a remediation link. It does not need network access.
</Check>

## Start local AWS services

```bash
make up
```

This starts LocalStack at `http://localhost:4566`, waits for the required services, and seeds DynamoDB, S3, SSM, and Secrets Manager placeholders.

<Note>
If Docker is unavailable, `make up` starts the Moto fallback on the same endpoint so backend development can continue.
</Note>

## Run the backend

In a second terminal:

```bash
make run
```

Verify the unauthenticated health check:

```bash
curl http://localhost:8010/health
```

## Run the frontend

In another terminal:

```bash
make run/web
```

The Vite dev server listens on [http://127.0.0.1:5173](http://127.0.0.1:5173).

## Make an authenticated call

Issue a local JWT and call `/me`:

```bash
TOKEN="$(make -s token)"
curl -H "Authorization: Bearer ${TOKEN}" http://localhost:8010/me
```

<Check>
The first call creates a local user record and returns `bootstrap.firstLogin: true`. A second call reuses the record and returns `bootstrap.firstLogin: false`.
</Check>

## Run tests

```bash
make test
```

For targeted checks:

```bash
make test/backend/unit
make test/backend/integration
make test/backend/contract
make test/frontend/unit
```

## Tear down

```bash
make down
```
