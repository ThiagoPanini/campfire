<p className="cf-kicker">AGENT POSTURE</p>

Campfire is built by a solo maintainer with heavy AI assistance. Agents are
treated as collaborators with strict review and a small, sharp toolbelt.

## Ground rules

- Agents read `AGENTS.md` and the active spec's `plan.md` before doing work.
- The [constitution](/sdd/methodology/constitution) wins over agent defaults
  whenever they conflict.
- All AI-generated changes are read and understood by the maintainer before
  being committed.

## Spec-driven by default

Features flow through the [Spec-Kit methodology](/sdd/methodology/spec-kit):
spec → plan → tasks → implementation. Agents follow the same lifecycle the
maintainer does.
