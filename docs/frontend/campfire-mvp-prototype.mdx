## Stack

The prototype is a root-level Vite, React 19, and TypeScript app with plain CSS. Mintlify docs live beside the app through `docs.json` and `docs/`.

## Run Workflow

```bash
npm install
npm run dev
npm run build
```

## Route Map

- `/`: public landing page.
- `/signup`: mock account creation and Google sign-up simulation.
- `/signin`: seeded account sign-in and Google sign-in simulation.
- `/onboarding`: protected preference flow.
- `/home`: protected member home.

Protected routes redirect to `/` when there is no in-memory mock user.

## Mock Session Model

Auth user and onboarding preferences are in React memory only. A full refresh clears user, preferences, and route state. Language and accent are the only values stored in `sessionStorage`, scoped to the current browser tab.

## Design References

The primary visual source is `specs/001-frontend-mvp-prototype/design-reference/project/Campfire Landing.html`. Root `DESIGN.md` is a synchronized fallback for home details not present in the Claude export.

## Component Structure

Routes and session actions are coordinated in `src/app/`. Shared UI primitives live in `src/components/`, fixed catalogs and copy in `src/data/`, screens in `src/screens/`, and visual tokens in `src/styles/`.
