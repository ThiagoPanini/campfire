The local backend workflow now lives in [Local environment](/guides/platform/local-environment). Use that page for the current LocalStack-backed `make` commands, debugger attach flow, and troubleshooting steps.

## Current commands

```bash
make doctor
make up
make run
make test
make down
```

Use `make token` when you need a local JWT for manual `/me` calls:

```bash
TOKEN="$(make -s token)"
curl -H "Authorization: Bearer ${TOKEN}" http://localhost:8010/me
```

<Warning>
Older references to DynamoDB Local on port `8000` and `scripts/backend-local-*.sh` are retired. LocalStack on `http://localhost:4566` is the canonical local AWS plane.
</Warning>
