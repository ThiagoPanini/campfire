# Platform

This section is the command-and-operations home for Campfire. It collects the local environment flow, the CI/CD contract, and the operational runbooks that go with them.

## Topic clusters

- Local environment: developer prerequisites, `make` commands, LocalStack usage, debugger attach, and troubleshooting.
- CI/CD: pull request gates, status checks, workflow structure, and extension guidance.
- Operations: deployment procedures, rollback guidance, environments, and incident handling.

## Current pages

- [Local environment](/guides/platform/local-environment)
- [LocalStack services](/guides/platform/localstack-services)

## Current implementation status

The local environment pages are current for `specs/002-cicd-local-env`. CI/CD, security, and operations pages are still planned in that spec and should be added as the corresponding Makefile targets and reusable workflows land.
