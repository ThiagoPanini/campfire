<p className="cf-kicker">CAMPFIRE · OVERVIEW</p>

Campfire is a private music hub for people who play in small, informal circles.
It is built around three jobs and nothing else.

<CardGroup cols={3}>
  <Card title="Track" icon="check">
    Record songs you already know how to play.
  </Card>
  <Card title="Learn" icon="book-open">
    Capture songs you are still working on.
  </Card>
  <Card title="Share" icon="users">
    Make that repertoire visible to your circle.
  </Card>
</CardGroup>

## Status

<span className="cf-badge">ALPHA</span> &nbsp; <span className="cf-label">CURRENT</span> Frontend MVP plus identity backend slice.

The frontend MVP is implemented. The first backend slice, `identity`, is also
implemented in code but still has manual verification gates open. Infrastructure
and CI/CD remain planned. See [Current status](/get-started/introduction/current-status)
for the canonical status matrix.
