---
title: "Install"
description: "Clone the repo and install backend dependencies."
---

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/ThiagoPanini/campfire.git
    cd campfire
    ```
  </Step>
  <Step title="Install backend dependencies">
    <Tabs>
      <Tab title="uv (recommended)">
        ```bash
        cd backend
        uv sync --extra dev
        ```

        `uv` creates `.venv/` and installs both runtime and dev extras (pytest, ruff, mypy) in one command.
      </Tab>
      <Tab title="pip">
        ```bash
        cd backend
        python -m venv .venv
        # Windows (Git Bash):
        . .venv/Scripts/activate
        # macOS / Linux:
        source .venv/bin/activate
        pip install -e ".[dev]"
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Copy the environment file">
    ```bash
    cp .env.example .env
    ```

    Defaults work out of the box. Variables are prefixed `CAMPFIRE_` and loaded by `pydantic-settings`.
  </Step>
</Steps>

<Check>
  Your venv is active when the shell prompt starts with `(.venv)`. If it does not,
  re-run the activate step above.
</Check>

Next: [Run the backend](/get-started/run-backend).
