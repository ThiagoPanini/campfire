Campfire is a Python FastAPI backend. To run it you need three things.

<Steps>
  <Step title="Python 3.11+">
    The project uses `datetime.UTC` and `X | Y` unions. Anything below 3.11 will fail at import.

    ```bash
    python --version
    ```

    Expected: `Python 3.11.x` or newer.
  </Step>
  <Step title="uv (recommended) or pip">
    `uv` is the project's package manager of choice — see the `uv.lock` at the repo root.

    ```bash
    pipx install uv   # or: pip install uv
    uv --version
    ```

    You can still use plain `pip` if you prefer — both paths are documented on the install page.
  </Step>
  <Step title="Git">
    To clone the repository.

    ```bash
    git --version
    ```
  </Step>
</Steps>

## Optional but useful
