<p className="cf-kicker">ADR · ACCEPTED</p>

Use PostgreSQL 16 as the only persistence engine for Campfire. Local
development runs `postgres:16-alpine`; future production infrastructure must
remain Postgres-compatible.

Related: [Data and persistence](/backend/data-persistence), [Database](/glossary/database).
