<p className="cf-kicker">ADR · ACCEPTED</p>

Use SQLAlchemy 2.x async with `asyncpg` for persistence adapters. The domain and
application layers stay independent of ORM types.

Related: [Backend architecture](/get-started/architecture/backend), [Repository pattern](/glossary/repository-pattern).
