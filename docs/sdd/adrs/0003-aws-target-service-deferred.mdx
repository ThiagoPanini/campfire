<p className="cf-kicker">ADR · SUPERSEDED BY [ADR-0011](/sdd/adrs/0011-aws-terraform-localstack-deprioritized)</p>

This ADR originally named AWS as the production target while deferring the
exact managed service. As of 2026-04-29 it is **superseded** by ADR-0011:
the deployment platform is now Render (see ADR-0010), and the AWS /
Terraform track has been deprioritized indefinitely.

The *application-level* constraints from this ADR — Postgres-only,
`asyncpg`, DSN via `SettingsProvider`, no session-scoped Postgres state,
allowlisted extensions only, `timestamptz` UTC — remain in force because
they preserve portability between any Postgres host.

Related: [ADR-0010](/sdd/adrs/0010-render-as-deployment-platform), [ADR-0011](/sdd/adrs/0011-aws-terraform-localstack-deprioritized), [ADR-0012](/sdd/adrs/0012-render-postgresql-as-production-database).
