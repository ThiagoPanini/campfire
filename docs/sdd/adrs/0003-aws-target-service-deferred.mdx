<p className="cf-kicker">ADR · ACCEPTED</p>

AWS is the production target, but the exact managed service is deferred to a
later infrastructure slice. The backend keeps an Aurora-ready Postgres posture
without baking a managed-service choice into application code.

Related: [Infrastructure](/get-started/architecture/infrastructure), [Roadmap](/get-started/introduction/roadmap).
