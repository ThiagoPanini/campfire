## Goal

Turn the Claude Design `campfire-v3` export into a clickable, responsive
prototype with mocked auth, onboarding, and home — five screens, no backend.

Spec 002 later replaced the auth/preferences mock boundary with a real identity
backend. The frontend visual scope from this spec remains the implemented app
surface.

## In scope

- Landing, Sign In, Sign Up, Onboarding, Home.
- Bilingual copy (EN / PT-BR).
- Five accent presets: `EMBER`, `FLAME`, `GOLD`, `COPPER` (default), `BRASS`.
- Mock session, mock seeded account, mock validation.
- Mintlify docs alongside the app.

## Out of scope

- Real OAuth, real persistence, real backend.
- Tweaks Panel from the Claude export.
- Repertoire / jam / group features (deferred to a later spec).
