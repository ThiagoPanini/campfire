## Status vocabulary

| Label | Meaning |
| --- | --- |
| `DRAFT` | The spec file still records draft status or open decisions. |
| `LIVE` | The slice is implemented and usable for its documented purpose. |
| `PARTIAL` | Implementation exists, but verification gates or documented scope remain open. |
| `PLANNED` | The area is intentionally deferred until a future spec. |
