---
title: Backend Local Dev
---

From the repository root:

```bash
docker compose up -d postgres
cd apps/api
uv sync
make migrate
make seed
make run
```

The API runs at `http://localhost:8000`. Check liveness with `/healthz` and
database readiness with `/readyz`.

Useful make targets:

| Target | Purpose |
| --- | --- |
| `make run` | Start FastAPI with reload |
| `make test-unit` | Run domain and use-case tests |
| `make test-integration` | Run real-Postgres integration tests |
| `make migrate` | Apply Alembic migrations |
| `make seed` | Re-apply the idempotent Ada seed |
| `make openapi-snapshot` | Regenerate the committed schema snapshot |
| `make check-aurora-extensions` | Fail on unsupported migration extensions |
